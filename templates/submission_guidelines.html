By using the example notebook provided in class, a predictions.txt file will be generated for you to submit. Please choose the correct phase according to your generations. Phases are divided into Validation/Test, Detection/Classification, MTA/USTC. Note that validation phases are private and the leaderboard scores are not shared.

Steps:

  1. Select a Phase.
  2. Upload your submission file (predictions.txt).
  3. Select Public/Private leaderboard submission option.
  4. Click submit.
